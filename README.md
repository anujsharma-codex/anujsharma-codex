<!-- HOW IT WORKS — KEPT AS CODE BLOCK FOR RECRUITERS -->
```
Input  →  location · panel specs · system size · segment (residential / commercial)
   ↓
Physical Layer  →  panel feasibility · orientation · shading analysis
   ↓
Energy Layer    →  daily / monthly / annual generation estimates
   ↓
Financial Layer →  ROI · payback period · LCOE · net cost after subsidy
```
