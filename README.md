## Featured Project

### [`solar-financial-calculator`](https://github.com/anujsharma-codex)

> A CLI tool for solar potential and financial analysis. Residential and commercial.  
> Config-driven. Real financial logic. Clean OOP with separated physical, energy, and financial layers.

**What it actually does:**

```
Input  →  location, panel specs, system size, segment (residential / commercial)

Physical Layer   →  panel feasibility, orientation, shading analysis
Energy Layer     →  daily / monthly / annual generation estimates
Financial Layer  →  ROI, payback period, LCOE, net cost after subsidy
```

**Architecture decisions:**
- All market rates, GST values, subsidy logic in one config file — change market conditions without touching logic
- Three class layers: physical ↔ energy ↔ financial — each responsible for one concern
- Separate handling for residential (MNRE subsidy) and commercial (accelerated depreciation, no subsidy)
